# scripts
Some useful scripts


## How to use from command line
wget https://raw.githubusercontent.com/mansehr/scripts/master/{script_name}  
chmod +x {script_name}  
nohup ./{script_name} & 

Or the direct way:  
  
curl -sL https://raw.githubusercontent.com/mansehr/scripts/master/{script_name} | sudo -E bash -
